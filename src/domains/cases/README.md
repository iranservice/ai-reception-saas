# Cases Domain

**Owner:** Cases module
**Responsibility:** Tickets, callback requests, escalation/de-escalation basics.

## Owns

- Support tickets
- Callback requests
- Escalation/de-escalation workflows

## Dependencies

- Identity, Tenancy, Authz, Conversations

## Anti-Patterns

- ❌ Do NOT put conversation messages here — that belongs in **Conversations**
- ❌ Do NOT put approval workflows here — that belongs in **Approvals**
